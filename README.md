## 项目介绍
>空闲时间动手搭建的一个框架，框架核心基于`spring boot 1.5.9`。

## 工程结构


## 集成模块
>Mysql：`5.7`
>
>缓存数据库：`redis`
>
>持久层：使用`Spring-data-jpa`让数据访问更简单、更优雅
>
>前端模板：`thymeleaf`
>
>日志管理：`log4j2`
>
>restful：使用`Swagger2`构建强大的RESTful API文档

## 其他
>构建工具: `Maven`
